# Slippi Web Bridge

This repository contains everything needed to stream to SpectatorMode.

There are 3 components to in this repository:
- The library `swb`
- The command-line application `cli`
- The graphical application `gui`

More information about each component can be found in the README in the appropriate directory.

Binary release downloads for both the CLI and GUI applications can be found under [releases](https://github.com/gcpreston/swb-rs/releases).
